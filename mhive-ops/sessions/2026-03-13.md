- Rolled back to stable US models (Sonnet 4.6 for planners, Grok Fast for researchers/doers)
- Gateway confirmed hot reload at 2026-03-13T16:07:51Z

## What Remains

- **Benchmark as independent task:** The framework is documented well enough (`mhive-ops/benchmarks/README.md`) for an independent agent to run benchmarks, score quality, and recommend model changes without touching the main openclaw codebase.
- **Higher token limit test:** Opus 4.6 was truncated at 2048 tokens on execution test. A re-run with 4096+ tokens would show whether Opus justifies its 5x cost premium over Sonnet.
- **New models to test:** When new models release (GPT-5, Gemini 2.0 Pro, Grok 3, etc.), run the benchmark before assigning to agents.
